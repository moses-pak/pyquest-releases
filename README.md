# PyQuest 🐍

**Learn Python at your own pace** — a game-like course built for data analysts
who already know SQL. Short lessons, live code you run in your browser,
auto-checked exercises, XP and streaks, and a real project at the end.

## Get started (2 minutes)

1. **[Download PyQuest.zip](../../releases/latest/download/PyQuest.zip)**
2. Right-click the downloaded zip → **Extract All…** → Extract. (Don't run it
   from inside the zip — extract first.)
3. Double-click the extracted **PyQuest.exe**. If Windows shows
   *"Windows protected your PC"*, click **More info → Run anyway** — the
   warning appears because the file isn't from a big-name publisher, not
   because anything is wrong. Still blocked? Right-click PyQuest.exe →
   **Properties** → check **Unblock** → OK, and run it again.
4. A small console window opens, then your browser opens the quest map.
   Keep the console window open while you learn; close it when you're done.

That's it. No installs, no setup — you need a browser and an internet
connection. Your progress saves automatically and picks up where you left off.

## Updating to a new version

Download the newest `PyQuest.zip` from the
[releases page](../../releases/latest), extract it, and replace your old
PyQuest.exe with the new one. **Your progress is kept automatically** — it
lives in your browser, not in the app file.

Tip: the **Export progress** button on the quest map saves a backup file.
**Import progress** restores it — even on a different browser or computer.
